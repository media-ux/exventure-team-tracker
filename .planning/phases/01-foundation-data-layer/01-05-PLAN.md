          tasks(count)

    Create src/hooks/useTasks.ts for task management:

    Export useTasks hook with:
    - `tasks` - array of tasks (optionally filtered by sub-unit)
    - `addTask(subUnitId, title, description, assignedTo?, dueDate?, status?)` - creates task
    - `updateTask(taskId, updates)` - updates any task field
    - `deleteTask(taskId)` - deletes task
    - `changeStatus(taskId, newStatus)` - convenience method for status updates
    - `reassignTask(taskId, newAssigneeId)` - convenience method for reassignment
    - `loading`, `error` - state management

    Fetch tasks with relations:
    ```typescript
    const { data, error } = await supabase
      .from('tasks')
      .select(`
        id,
        title,
        description,
        status,
        due_date,
        created_at,
        updated_at,
        sub_units(id, name, projects(id, name)),
        assigned_to_member:team_members!assigned_to(id, name, role),
        created_by_user:auth.users!created_by(id, email)
      `)
      .order('created_at', { ascending: false })
    ```

    Update task implementation:
    ```typescript
    const updateTask = async (taskId: string, updates: Partial<Task>) => {
      const { data, error } = await supabase
        .from('tasks')
        .update({
          ...updates,
          updated_at: new Date().toISOString()
        })
        .eq('id', taskId)
        .select()
        .single()

      if (error) throw error
      return data
    }
    ```

    IMPORTANT: All mutating operations (add, update, delete) should refetch data or update local state to keep UI in sync.
